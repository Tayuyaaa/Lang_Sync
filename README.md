# Lang_Sync
# 🌐 Tab Lang Memory (Windows)

Блискавичне розширення для Chrome, Edge та Brave, яке **запам'ятовує розкладку клавіатури (Українська / Англійська) для кожної окремої вкладки** і автоматично перемикає її при переході. 

Більше ніяких `ghbdtn` замість `привіт` після перемикання з YouTube на робочу пошту.

## 🚀 Чому це круто (Технічні деталі)
Більшість подібних програм висять у фоні 24/7 або використовують повільний PowerShell. Цей проект працює інакше:
* **Швидкість 1 мілісекунда:** Використовує нативний скомпільований `C#` мікро-файл (.exe на 4 КБ).
* **0% фонового навантаження:** Процес не висить у Диспетчері завдань. Він запускається системою Chrome (Native Messaging) лише на мить під час зміни вкладки і миттєво "вмирає".
* **Довгострокова пам'ять:** Використовує `chrome.storage.local`, тому не забуває мову вкладок, навіть якщо браузер переводить їх у сплячий режим (Memory Saver).
* **Жодного сміття:** Установник сам компілює код під твій ПК і прописує рівно один ключ у реєстр.

## 🛠 Як встановити (Займе 1 хвилину)

### Крок 1: Додаємо в браузер
1. Завантаж архів з останнього релізу в розділі **[Releases]** (справа на цій сторінці) та розпакуй його в зручне місце. *(Увага: не переміщуй папку після встановлення!)*
2. Відкрий браузер і встав у адресний рядок: `chrome://extensions/`
3. У правому верхньому куті увімкни **Режим розробника** (Developer mode).
4. Натисни **Завантажити розпаковане** (Load unpacked) і вибери папку `MyLangExt`.
5. Скопіюй **ІДЕНТИФІКАТОР (ID)** розширення (довгий набір літер).

### Крок 2: Зв'язуємо з Windows
1. Зайди в розпаковану папку і запусти `install.bat`.
2. У чорному вікні встав свій скопійований ID і натисни Enter.
3. Скрипт миттєво створить міст між браузером і системою.

**🔥 ВАЖЛИВО:** Після встановлення повністю закрий браузер і відкрий його знову. Усе працює!

---

## 🗑 Як видалити
Якщо захочеш прибрати розширення:
1. Видали його на сторінці `chrome://extensions/`.
2. Запусти `uninstall.bat` з тієї ж папки (це безслідно очистить запис у реєстрі Windows).
3. Видали саму папку.

---
*Примітка: Наразі скрипт налаштований виключно на Українську (0x0422) та Англійську (0x0409) розкладки клавіатури.*
